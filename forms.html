Forms:

A PHP form is an HTML element that gathers user input (such as text or selections) on a web page. When a user submits the form, the data is sent to a PHP script on the server for processing. Forms are crucial for interactive websites, allowing users to interact by inputting data. PHP scripts handle the submitted data, perform validations, and generate responses for users. In essence, PHP forms enable the exchange of data between users and web servers, powering dynamic web applications.

PHP Programs and Outputs

Post Method

When you use the "POST" method in a form, the data entered by the user is sent in the HTTP request body. This method is suitable for sending sensitive information or large amounts of data because the data is not visible in the URL.

Get Method

When you use the "GET" method in a form, the form data is appended to the URL in the form of query parameters. This makes the data visible in the URL, which can be seen in the browser's address bar.

Super Global Variable

In PHP, superglobals are special global variables that are available in all scopes throughout a script. They are predefined variables that contain useful information and are used to gather form data, manipulate sessions, handle cookies, and perform other essential tasks in web development. Superglobals are prefixed with a dollar sign ($) and an underscore (_).

Example of some Super Global Variables:


  • $_GET: Used to collect form data after submitting an HTML form with the HTTP GET method. Gathers data from the form's query parameters appended to the URL.

  • $_POST: Used to collect form data after submitting an HTML form with the HTTP POST method. Gathers data from the form fields sent in the request body.

  • $_REQUEST: Can be used to collect form data regardless of the HTTP method used (GET or POST). Gathers data from both the $_GET, $_POST, and $_COOKIE superglobals.
